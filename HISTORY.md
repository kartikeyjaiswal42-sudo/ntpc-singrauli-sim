# ntpc-singrauli-sim/ — full history

Educational simulation of an NTPC Singrauli Stage-2 500 MW coal unit, built from a real site photo walk-through. **Live: https://kartikeyjaiswal42-sudo.github.io/ntpc-singrauli-sim/** (GitHub Pages, repo `kartikeyjaiswal42-sudo/ntpc-singrauli-sim`, auto-deploys on push to `main` via `.github/workflows`). Vite multi-page app (`index.html` landing + `2d/` photo console + `illustrated/` + `3d/`), dep `three`. **Deploy = just `git push`; do NOT run localhost** (user instruction). Dev port 3005 (`./start.sh`).
